Features:
- home screen that includes all the available songs on your phone. you can share/add to playlist/remove songs.
- create/remove playlists.
- albums screens.
- and all tha basic expectations of a music player.

Instalation:
```
- clone the repo
- flutter pub get
- flutter run
```
